# Champion Images

Place champion PNG images in this folder with the following naming format:

## Naming Convention
- Remove spaces and special characters
- Use PascalCase
- File extension: `.png`

## Examples
- Aatrox.png
- AurelionSol.png
- Belveth.png
- Chogath.png
- DrMundo.png
- JarvanIV.png
- Kaisa.png
- Khazix.png
- KogMaw.png
- Leblanc.png
- LeeSin.png
- MasterYi.png
- MissFortune.png
- Nunu.png (for Nunu & Willump)
- RekSai.png
- Renata.png (for Renata Glasc)
- TahmKench.png
- TwistedFate.png
- Velkoz.png
- XinZhao.png

## Image Specifications
- Format: PNG
- Recommended size: 120x120px or larger
- Square aspect ratio preferred
- Transparent background optional

## Where to get images
You can download champion images from:
- Data Dragon (Riot's official CDN)
- Community Dradon
- League of Legends Wiki
- Or create custom icons

If an image is missing, a placeholder will be shown instead.
